# Getting Started

## Prerequisites

- Node.js 18+
- npm 9+

## Install Dependencies

`ash
npm ci
`

## Run Local Dev Server

`ash
npm run docs:dev
`

Default local URL is typically:

`	ext
http://localhost:5173
`

## Build for Production

`ash
npm run docs:build
`

The output directory is:

`	ext
docs/.vitepress/dist
`

## Preview Production Build

`ash
npm run docs:preview
`
